# 🚗 Vehicle Data Analysis App

This project is a web application built with Streamlit for analyzing vehicle data.  
It allows users to explore and visualize information such as price and mileage through interactive filters and charts.

## 📊 Features

- Filter vehicles by price range
- Filter vehicles by mileage (odometer)
- Interactive histogram for mileage distribution
- Scatter plot showing relationship between price and mileage
- Summary metrics (total vehicles, average price, average mileage)

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
